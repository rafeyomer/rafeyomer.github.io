Insurance Cost Analysis — Case Study

Goal: Predict annual charges using demographic/lifestyle variables.

Approach: Data cleaning (missing checks, outlier review), EDA (distributions, pairwise plots), multiple linear regression with interaction terms (e.g., smoker×age), k-fold CV.

Results: R² ≈ 0.82, RMSE ≈ $5k; smoking emerged as strongest driver (~$22.6k).

Next: Try regularization (Lasso/Ridge), assess non-linearities (splines), and fairness checks.

Normal Distribution Interactive App — Case Study

Goal: Make Normal distribution parameters and probabilities intuitive.

Approach: R Shiny UI with sliders for μ, σ, interval bounds; reactive plots via ggplot2/plotly; shaded probability regions; live computed P(a<X<b), P(X≤a), P(X≥b).

Results: Interactive demo used for coursework and quick probability sanity checks.

Next: Add t & χ² distributions, export images, and a “share state” URL.
